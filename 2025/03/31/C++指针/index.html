C++指针

指针的定义及作用

指针的定义：

首先，要学习指针，我们肯定需要明白指针是什么：指针是一个变量，其值为另一个变量的地址，即内存位置的直接地址。因为是变量，所以指针在存储其他变量地址之前，也需要先声明。指针变量的一般声明形式为：

其中，type是指针变量类型，var_name是指针变量名。通过*来声明该变量为指针变量。

不同数据类型变量的指针的值都是一个代表内存地址的长的十六进制数。

指针的作用：

指针能使得不同区域的代码可以轻易的共享内存

指针使得一些复杂的链接性的数据结构的构建成为可能，有些操作必须使用指针，比如申请堆内存，还有C++或者C语言中函数的调用分为值传递和引用传递，如果要在函数中修改被传递的对象，就必须通过这个对象指针来完成（引用的底层也是通过指针实现的）。（值得一提的是java函数的调用只有引用传递没有值传递）

指针的使用：

这里我们定义了一个int型变量a，并将a的值赋为0。之后我们定义了一个int类型的指针，并取a的地址赋给x（&为取值符，会返回变量的地址），此时指针x指向的就是a的地址。然后我们将x进行解引用操作（即通过指针指向的地址拿到变量具体存储的值），将10赋给该变量。通过以上操作，最后打印出来a的值为10。以上就是指针的最简单的应用，通过指针我们能轻松的对内存进行操作，进而改变变量的值。

值得注意的是，* 既能表示声明变量为指针变量，又能表示解引用（即获取变量具体的值）；同样 & 既能表示取地址，又能表示声明引用。那我们该如何区分它们呢？ 区分*的关键在于它是否用于变量的声明和定义，即是否跟在变量类型后面（int *x 此时*表示声明x为指针变量），当它直接作用于变量前面（int y = *x 此时*表示的是解引用，即取x指向变量的值赋给y）； 同理&也是如此，但要注意的是&作为取地址时不能放在等号（=）左边，因为变量的地址是常量值不能被改变。

空指针和野指针

  • 空指针：即指针指向的内存地址为空，int * x = nullptr；此时x为空指针
  • 野指针：指针在初始化时未赋值直接使用，或者使用指向内存已被释放的指针，此时指针指向的内存地址是未知的，这是非常危险的操作，会导致程序崩溃以及其他未知问题，所以在程序中应该避免野指针的产生。声明指针的时候立即初始化为nullptr（int * p = nullptr）;使用delete或free释放内存后立即将指针置为nullptr（delete p；p = nullptr）。

指针运算

指针运算都是作用在连续存储空间上才有意义。

（1）指针加减整数运算
int x[10]; int n = 3; int * p = &x[5];
p + 1 //指向内存空间中x[5]后面的第1个int 型存储单元
p + n //————————–后n个
p - 1 //——————-前面1个
p - n //——————-前面n个

（2）指针变量自增自减运算
int x[10]；int * p = &x[5];
p++ //p指向x[5]后面的第1个int型内存单元
++p //同上
p– //p指向x[5]前面的第1个int型内存单元
–p //同上

(3)两个指针相减运算
设p1, p2是相同类型的两个指针，则p2 - p1的结果是两支针之间对象
的个数，如果p2指针地址大于p1则结果为正，否则为负
int x[5], * p1 = &x[0], * p2 = &x[4];
int n;
n = p2 - p1;//n 的值为4 即为他们之间间隔的元素的个数
运算方法：（p2储存的地址编码-p1储存的地址编码）/4 若是double类型则除以8 char类型除以1

（4）指针的运算关系
设p1、p2是同一个指向类型的两个指针，则p1和p2可以进行关系运算，
用于比较这两个地址的位置关系即哪一个是靠前或者靠后的元素
int x[4], * p1 = &x[0], * p2 = &x[4];
p2 > p1; //表达式为真

关键区分

  • 通过中文区分

    • 常量指针 ：指向常量的指针（数据是常量，指针不是常量）

    • 指针常量 ：指针本身是常量（指针是常量，数据不是常量）

  • 通过代码区分 （通过const位置区分）

    • 常量指针 ：const在数据类型左边（const int* ptr），const修饰数据类型，所以数据是常量无法修改
    • 指针常量：const在数据类型右边（int* const ptr）,const修饰指针变量（ptr），所以指针是常量无法修改

指针与数组

数组的地址是在一整块连续的空间上，我们可以通过&拿到对应元素的地址，数组名可以直接表示为数组的首地址，数组名是一个指针常量

由于数组的元素地址是规律递增，所以我们能够通过指针方便地访问数组中的元素

因为数组名a等价于数组首地址，将a赋给指针p，所以a[i] 等价于 p[i]，都相当于 *(a+i) 或者*(p+i)

值得注意的是，二者的区别为：

  • sizeof(a)：返回整个数组的字节数（10 * sizeof(int)）。
  • sizeof(p)：返回指针变量的大小（通常 4 或 8 字节）。

由于a为指针常量所以无法被重新赋值或者进行自增自减运算，使用p则可以进行相应的操作
